## Projects

### [Interactive Expense Manager](./week-1/expense-manager/)
A modular Rust application built in stages to practice:
- Interactive CLI loops
- HashMap data structures
- Code modularization
- Error handling and user input validation

### [SEP-41 Token](./week-2/contracts/sep-41-token/)
A Soroban smart contract implementing the SEP-41 token interface, including:
- Minting and burning
- Transfer and transfer_from
- Allowance approval system

### [School Management System](./week-3/contracts/school-management/)
A Soroban smart contract for managing school records, including:
- Registering students
- Updating student class
- Recording and retrieving payment history
- Removing students
